# frontend/css

Shared styling architecture for CARA.

| File | Responsibility |
| --- | --- |
| `tokens.css` | Color, spacing, typography, radius, shadow, and timing variables |
| `base.css` | Reset, body defaults, semantic typography, accessibility helpers |
| `components.css` | Buttons, cards, badges, chips, forms, modals, tables, toasts |
| `nav.css` | Public navbar, ward sidebar/topbar, compass bottom nav |
| `animations.css` | Keyframes, reveal utilities, reduced-motion rules |
| `pages/landing.css` | Landing page layout |
| `pages/about.css` | About page layout |
| `pages/research.css` | Research page layout |
| `pages/join-us.css` | Join Us page layout |
| `pages/privacy.css` | Privacy page layout |
| `pages/ward.css` | Ward dashboard layout and responsive behavior |
| `pages/compass.css` | Compass dashboard layout and responsive behavior |
| `style.css` | Legacy import aggregator |
